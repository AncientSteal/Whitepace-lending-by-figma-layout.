# Whitepace Лендинг по макету Figma.
Проект представляет собой классический одностраничный сайт о продукте Whitepace, реализованный с использованием классических инструментов веб-разработки.

## ❎ Структура проекта

```
├── images/ изображения
├── vendor/ # папка шрифтов
├── js/ # скрипты JavaScript
├── styles/ # папка стилей
├── index.html # главный файл со структурой сайта
└── README.md
```

## 🚀 Основные возможности

* Удобное меню для навигации по сайту
* Отзывчивый интерфейс
* Формы с кастомной валидацией
* Имитация ответа от сервера реализована через *alert-сообщения*
